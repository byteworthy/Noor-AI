### iOS Submission Steps

1. **App Store Connect Setup**
   - Create app record
   - Fill all metadata fields
   - Upload screenshots
   - Add privacy manifest

2. **Build Upload**
   - Archive in Xcode
   - Upload via Xcode Organizer or Transporter
   - Wait for processing (10-30 min)

3. **TestFlight Beta (Optional)**
   - Add internal testers (up to 100)
   - Get feedback
   - Fix bugs

4. **Submit for Review**
   - Select build
   - Add review notes (if needed)
   - Submit

5. **Review Time**
   - Typical: 24-48 hours
   - Check status in App Store Connect
   - Respond to any questions quickly

6. **Approval & Release**
   - Choose: Manual release or automatic
   - Monitor reviews & ratings
   - Respond to user feedback

---

### Android Submission Steps

1. **Google Play Console Setup**
   - Create app
   - Complete store listing
   - Upload graphics assets

2. **Release Track Selection**
   - Internal testing (up to 100 testers)
   - Closed testing (larger beta)
   - Open testing (public beta)
   - Production (full release)

3. **Upload APK/AAB**
   - Build signed AAB (recommended)
   - Upload via Play Console
   - Set rollout percentage (start with 10%)

4. **Content Rating**
   - Complete IARC questionnaire
   - Get rating certificate
   - Confirm rating

5. **Pricing & Distribution**
   - Select countries
   - Set pricing (or free)
   - Enable in-app purchases

6. **Review & Publish**
   - Submit for review
   - Review time: 1-7 days (typically 1-2)
   - Monitor for issues

7. **Rollout Management**
   - Start at 10% rollout
   - Monitor crash reports
   - Increase to 50% â†’ 100% over 2-3 days
